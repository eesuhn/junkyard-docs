<Note>
  ### Note

  - Both the borrow and base APY are sourced real-time from Loopscale.
  - You may obtain the Loop ID (`loop_id`) from the [/loopscale/loop/markets](/reference/loopscale/get-loop-markets) endpoint.
  - Use `include_mint_addrs=true` to include mint addresses in the `include` object of the response.
  - Access is limited to authenticated users; reach out to [our team](https://eesuhn.github.io/eesuhn/) for free API keys.
</Note>
